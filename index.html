Ready? Let’s get started.

  • Do you have a GitHub account?

    • I need a GitHub account!
    • I already have a GitHub account.

  • Create a GitHub account

    Head over to the GitHub homepage and create an account (if you don't have one already).

  • What's your GitHub username?

  • Create a repository

    Once your account is created and you're logged in,
    create a new repository

    Make sure the Repository name is username.github.io

    Also make sure to check the Initialize this repository with a README checkbox!

  • Create a new file

    This will be the first page people see when they visit your website.

    Click the "Create new file button"...

  • Create index.html

    Name your file index.html

    In the editing area, type in some content.

    HELLO WORLD!

  • Save your changes!

    Saving changes to a repository is called "committing". Click the Commit changes button.

  • …and you're done!

    Head on over to http://username.github.io. Give it up to ten minutes for your page to show up—there will be a delay this very first time. Any future changes will show up quite a bit faster.
